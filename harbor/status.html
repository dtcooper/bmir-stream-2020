AutoDJ Requests

AutoDJ Requests

Make a request to play next. This will be the next track the AutoDJ selects, inserting into the top of the queue. (Unfortunately, the only way to see if it worked is to listen.)
Types of requests:
  • A full URL to an audio file to request, eg.
    https://www.example.com/file.mp3
  • A video from youtube (or similar) prefixed withyoutube-dl:, eg.
    youtube-dl:https://www.youtube.com/watch?v=dQw4w9WgXcQ
  • Computerized speech, prefixed with say:, eg.
    say:Hi mom!
  • Read more about liquidsoap request formatting .



Restart

Restart Harbor

Notes:
  • Current DJ will be disconnected
  • AutoDJ will be restarted
  • However, listeners WILL NOT be disconnected

Reboot Server

Notes:
  • Server will be completely restarted
  • Current DJ will be disconnected (if any)
  • All listeners will be disconnected
  • You probably DON'T want to do this unless things are badly broken
